# Share It

## О проекте
Приложение «Share It» позволяет пользователям сдавать личные вещи в аренду.

## Из чего состоит
Приложение состоит из двух модулей:
- Share It Server
- Share It Gateway

## Share It Server
Содержит всю основную логику приложения.

## Share It Gateway
Отвечает за валидацию запросов от пользователей.

## Стек технологий
- Java
- Spring Boot
- PostgreSQL
- Maven
- Docker
